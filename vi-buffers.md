**note**/

  * : **b [Num]** -- move to that buffer (window)
  * : **ls** -- shows buffers
  * : **[N] bd!** -- unload buffer, changes lost
  * : **[N] bd** -- if buffe was changed, this fails
  * : **% delete** -- delete all buffers
  * : **[N] sb** -- split window and edit buffer
  * : **Ctrl-w** -- followed by h, j, k, or l - switch pane
